# Jewellery Landing Page - Project 1

### Overview
This project is a premium, minimalist Jewellery Landing Page developed as part of the DecodeLabs Internship (Project 1). The website uses semantic HTML structure and custom CSS layouts to simulate a luxury, high-end editorial digital storefront for a fine jewelry brand called "SILK & STONE".

It is designed to strengthen understanding of:
* Semantic webpage structure
* CSS layouts and structural design resets
* Interactive hover animations and effects
* Responsive web design using media queries

### Objective
To build a landing page that can:
1. Display a luxury brand identity with clean typography
2. Organize fine jewelry product modules efficiently
3. Scale layout elements dynamically across screen sizes
4. Provide clean link structures for anchor navigation

### Technologies Used
* HTML5
* CSS3
* Google Fonts API

### Features
* Fixed navigation header with smooth anchor scrolling
* Premium editorial color palette (off-white, sangria, and rosewood)
* Four-column product grid system displaying distinct categories
* Interactive micro-animations (soft luxury zoom on card hover)
* Fully responsive layout scaling for tablet and mobile devices

### Concept Used
* Flexbox layout grid wrapping for structural alignment
* Box-sizing resets for accurate dimension calculations
* CSS transition properties for smooth dynamic animations
* Media queries for screen size breakpoints
* Relative path linking for folder-independent asset tracking
