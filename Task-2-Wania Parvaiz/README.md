# 🍝 Pasta & Pine | Artisan Pasta Laboratory

### Overview
This project is a simple and clean landing page for a fictional premium pasta brand called “Pasta & Pine.” It is created as part of the DecodeLabs Internship program. The aim is to design a modern, minimal, and luxury-style food website.

The project focuses on:
* Building a strong and premium brand identity
* Using clean fonts and natural colors
* Creating a structured and responsive layout

---

### Objective
The goal of this project is to build a landing page that:
* Shows a premium brand using soft and natural colors
* Displays artisan food products in clear sections
* Works smoothly on all screen sizes
* Provides easy navigation through the page

---

### Technologies Used
* **HTML5** for structure
* **CSS3** for styling and layout
* **Google Fonts** for elegant typography

---

### Features
* **Sticky Navigation:** Menu stays visible while scrolling.
* **Hero Section:** A strong introduction with a call-to-action button.
* **Responsive Recipe Cards:** Layout changes from stacked on mobile to side-by-side on desktop.
* **Elegant Color Theme:** Uses natural colors like green and brown for a premium look.
* **Hover Effects:** Simple effects when moving the mouse over elements.

---

### Concepts Used
* **Flexbox** for layout and alignment
* **Sticky positioning** for navigation
* **Object-fit** for proper image display
* **Media queries** for responsive design
* **Semantic HTML** tags like `header`, `section`, `nav`, and `footer`

---

### How to Run
1. Download or clone the project folder.
2. Open it in a code editor like VS Code.
3. Open `index.html` in a web browser.
