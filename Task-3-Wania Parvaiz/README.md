# 🎨 Color Palette Generator

### Overview
This project is a simple and interactive Color Palette Generator. It helps users create color combinations quickly for design inspiration. 

It has a clean interface where users can:
* **Generate** random color codes
* **Lock** colors they want to keep
* **Copy** HEX color values easily

This project helps in learning:
* JavaScript DOM manipulation
* Dynamic styling and class changes
* CSS Flexbox layout
* User interactions and clipboard usage

---

### Objective
The goal of this project is to build a tool that can:
* Generate random HEX color codes
* Lock selected colors so they do not change
* Show feedback messages like “COPIED!” when a color is copied
* Work properly on desktop and mobile screens

---

### Technologies Used
* **HTML5** for structure
* **CSS3** for styling
* **JavaScript (ES6)** for functionality

---

### Features
* **Generate Colors:** Create new color palettes with one click or Spacebar.
* **Lock Colors:** Keep selected colors unchanged during generation.
* **Copy Colors:** Copy HEX codes with one click.
* **Responsive Design:** Works on all screen sizes.
* **Visual Feedback:** Shows simple animations and messages for better user experience.

---

### Concepts Used
* **Flexbox** for layout and alignment
* **JavaScript DOM API** for updating page content
* **Event Listeners** for clicks, keyboard input, and copy actions
* **Media Queries** for responsive design

---

### Project Structure
The project contains HTML, CSS, and JavaScript files organized in a single folder.

---

### How to Run
1. Download or clone the project folder.
2. Open it in VS Code or any code editor.
3. Open `index.html` in a web browser.
4. *(Optional)* Use Live Server for better experience.
